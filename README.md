# Mega Sena Resultado

Retorna último resultado da Mega Sena

## Pré Requisitos

Instalar Node.js

```
https://nodejs.org/en/download/
```

## Instalação

```
git clone https://github.com/claudioonohara/mega_sena_resultado.git
```

```
cd mega_sena_resultado
```

```
npm i
```

## Execução

```
npm start
```
